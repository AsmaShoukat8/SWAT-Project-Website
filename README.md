# SWAT Project Website Clone

This project is a front-end recreation of the official SWAT (Sindh Water and Agriculture Transformation) Project website.

The project was developed as part of my internship/work assignment to practice and demonstrate skills in:

- HTML5
- CSS3
- Bootstrap 5
- Responsive Web Design
- CSS Animations
- Interactive UI Components

## Disclaimer

This website was originally designed and developed by the organization/company.

My role was to recreate the front-end interface for learning, practice, and skill development purposes. All project content, branding, images, and information belong to their respective owners.

## Features Implemented

- Responsive navigation bar with multi-level dropdowns
- Animated hero section
- Partner cards
- About and Objectives sections
- Key Features cards with hover effects
- Civil Works and HAI Program sections
- Events carousel
- Project timeline
- GRM (Grievance Redressal Mechanism) complaint form

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- Animate.css

## Learning Outcomes

Through this project I practiced:

- Complex layouts using Bootstrap Grid
- CSS animations and transitions
- Responsive design principles
- Multi-level dropdown menus
- Custom timelines
- Image carousels
- Glassmorphism effects
- UI recreation from an existing design
